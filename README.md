puppet-ticket-face
==================

Description
-----------

A Puppet Face for creating Redmine tickets via the Faces API.

Requirements
------------

* `puppet ~> 2.7.0`
* A [Puppet Labs Redmine account](http://projects.puppetlabs.com).

Installation
------------

Install puppet-ticket-face as a module in your Puppet master's module
path.
